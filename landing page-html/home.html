Our Inspiration

Our target is to create book rental system for school and university students. Book vendors throughout city will post book details for renting availability. Students(buyer) can browse for book on website through subject and standard filter. Brighter side of E-books : lightwieght, more flexible than paper books, flexibility of changing font size hightlight and commenting text neatly

Features
Wondering what's on the plate?

Login/SignUP/Order Placed Confirmation User will receive token for activities like Registration, Login and After Order is successfully placed.
Wide range of database Diffrent types of book categories like : Entrance Exam,Academic & Professional,Biographies & Auto Biographies,Children & Teens,Regional Books,Business & Management,Health and Cooking etc.
Search Filters To find books of choice user can use search filters like: Book category, Book publication, Book author.
Price Filters User can apply price filters like "Price : High to Low" and "Price : Low to high". Also User can apply filters to book like "Most polpular book" in any book category.
Add to Cart User can order more than one book at a time and also when User buy same book in quantity greater than one cost will automatically calculated. Also User will receive a token of successfully placing an order.
Have doubts? - send us mail User can Send us a mail and our help desk team will try to solve the difficulties in as soon as possible.
